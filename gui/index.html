Instructions:

1. If you haven't already before launching the script, you must insert your required information into the script settings tab

2. Then click on save to update the information across the application

Note: This step won't be required once a save is detected

3. Once done, your able to use the scripts tab and begin gathering driving test availability

4. If you would like to be one of the first to identify cancellations, enable auto mode with an appropriate time interval

5. If you would like this to run by itself when you are away or not wanting the app on the screen, click on set scheduling with a time slot in mind
